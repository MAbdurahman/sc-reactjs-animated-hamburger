# sc-reactjs-animated-hamburger

<p>
sc-reactjs-animated-hamburger is a simple ReactJS navbar with an animated menu button 
made with style-components, react-router-dom, and react-helmet. The navbar and menu
button are styled with style-components, links created with react-router-dom to
navigation between the pages, and to manage the document head/title with react-helmet.
</p>

## Features

- Responsive design
- useState hook
- useReducer hook
- useContext hook
- custom hooks
- custom notification
- custom validation
- Style-components
- React-router-dom
- React-helmet

### To install dependencies
```shell
npm install
```
or
```shell
yarn or yarn install
```
### To run the app
```shell
npm run dev
```
or

```shell
yarn dev
```

### Build the app
```shell
npm run build
```
or

```shell
yarn build
```
